# Lojinha de Afiliados

Loja de afiliados integrada ao Google Sheets.

## Sobre o projeto

Este projeto é uma aplicação web simples que permite que criadores de conteúdo disponibilizem seus links de afiliados de forma organizada, no estilo de uma loja.

O Google Sheets funciona como o back-end da aplicação, onde os produtos são cadastrados e gerenciados.

## Funcionalidades

- Listagem dinâmica de produtos
- Busca por nome
- Filtro por categorias
- Links para múltiplas lojas por produto
- Layout responsivo

## Tecnologias utilizadas

- HTML
- CSS
- Tailwind
- JavaScript
- Google Sheets
